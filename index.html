About this project

This is a design from Frontend Mentors that I hand coded and turned into a WordPress theme. Frontend Mentors provided the Figma files for the home, destination, crew and technology pages. I added the archive, 404 and blog post pages to turn this into a fully functioning WordPress theme.

This was my first time creating a WordPress theme from scratch. I wanted to challenge myself by not using any frameworks, while still including all of the needed features to have a functioning WordPress theme. I learned a lot during this project will allowed me to take on larger theme builds.

About this project

This website was built for the owners of Jefferson Inn in Whitehall, MT. They had an old, outdated website that was not user friendly or easily maintainable. The website uses WordPress and was built using Oxygen Builder.

The goal for the website was to allow customer's to book rooms and to also promote other business in and around Whitehall, MT. The owners had built relationships with companies that offer outdoor activies, sports and entertainment. I built out sections to allow customers to learn about nearby activies to help them find things to do based on their interests.

This website was built for the owner of Jefferson Inn in Whitehall, MT. They had an old and outdated website that needed to be mobile optimized and help with promoting other local businesses.
