Pour Wine & Taproom

Technology Used

This is a fully responsive website with a focus on the moblie design. I intergrated Tockify on the events page so they could easily update it. I also link the beer menu to Untappd, a beer review application that allows them to keep their beer menu updated.

  • HTML
  • CSS
  • JavaScript
  • Bootstrap 4
  • Netlify
  • GitHub

About this project

Pour Wine & Taproom is a new local pub in Mesa, Az. I had gone there a few times and noticed they didn't have a website built. I developed the website before presenting it to the owners. They enjoyed the simple design and the mobile responsiveness.

After the inital design I added an events page to highlight everything they have going on at Pour. I also created a merchandise section to show off their apparel.
